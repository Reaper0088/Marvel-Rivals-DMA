Marvel Rivals (DMA) is an experimental project focused on **DMA-based tooling and external system interaction**.
This repository contains documentation and example components related to working with **DMA devices and secondary processing systems**.

---

## 📌 Overview

This project explores concepts such as:

* ⚙️ External system communication
* 💻 DMA hardware workflows
* 🖥️ Multi-PC processing setups
* 🧩 Software architecture for external tools

The goal of the project is to experiment with **low-level system interaction and hardware-assisted workflows**.

---

## ✨ Features

### 🎯 Aim System

* FOV Circle
* Hardlock Targeting
* Adjustable Smoothing
* Bone Selection

### 👁️ Visuals

* Player ESP: Box, Corner Box, and customizable overlays
* Health Display: Dynamic color-coded health bars
* Information Display: Player Name and Distance
* Snaplines: On-screen and off-screen indicators
* FOV Arrows: Directional indicators for visible and off-screen targets
* Dynamic Health Coloring based on current HP

### 🎨 Visual Customization

* Custom Health Bar Colors
* Adjustable Box Thickness
* Adjustable Box Roundness
* Fuser Mode Support

### 💾 Configuration System

* Save Configurations
* Load Configurations
* Automatic Config Loading on Startup

### 1️⃣ Download the Files
